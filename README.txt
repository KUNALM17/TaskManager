========================================================================
🚀 PREMIUM TASK MANAGER - FULL STACK APPLICATION
========================================================================

Welcome to the Premium Task Manager! This application is a complete
Full-Stack solution built with Spring Boot (Backend) and React (Frontend).

------------------------------------------------------------------------
✨ KEY FEATURES
------------------------------------------------------------------------
*   Role-Based Access Control (Admin & User roles)
*   Interactive Kanban Board with Drag-and-Drop
*   Real-time Dashboard Statistics with Progress Visualization
*   Magic Checklist: Automatically completes tasks when all items are checked
*   File Attachments & Management
*   CSV Report Exporting for all tasks
*   JWT-based Secure Authentication

------------------------------------------------------------------------
🛠️ TECH STACK
------------------------------------------------------------------------
*   Backend: Spring Boot 3, Spring Security, Hibernate, PostgreSQL
*   Frontend: React, Vite, Tailwind CSS, Framer Motion, Recharts
*   Deployment: Railway (Backend/DB), Vercel (Frontend)

------------------------------------------------------------------------
🚀 LOCAL SETUP
------------------------------------------------------------------------
1.  Backend:
    *   Navigate to /backend
    *   Run: ./mvnw spring-boot:run
    *   Starts at: http://localhost:8080

2.  Frontend:
    *   Navigate to /frontend
    *   Run: npm install
    *   Run: npm run dev
    *   Starts at: http://localhost:5173

------------------------------------------------------------------------
🌍 DEPLOYMENT NOTES
------------------------------------------------------------------------
*   Ensure VITE_API_URL is set in your environment variables.
*   The application is already configured with railway.json and Dockerfile
    for cloud deployment.

Enjoy managing your tasks with style!
